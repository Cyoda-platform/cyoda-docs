The JSON Schema reference is generated at build time from the schemas
vendored under `vendored/schemas/`. Each schema gets its own viewer page,
grouped by category. Use the sidebar to navigate the generated subsections.

A downloadable zip of all schemas is available at
[`/schemas.zip`](/schemas.zip).
