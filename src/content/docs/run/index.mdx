Pick your packaging. Every tier runs the same app.

- [Desktop](./desktop/) — single binary, in-memory or SQLite.
- [Docker](./docker/) — bespoke integrations and composed environments.
- [Kubernetes](./kubernetes/) — Helm-based production deployment on PostgreSQL.
- [Cyoda Cloud](./cyoda-cloud/) — hosted offering (test/demo today; commercial SLA coming).
